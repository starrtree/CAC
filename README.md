# CAC Website

Premium static website concept for The Cincinnati Air Conditioning Company.

## Current build

This repo currently contains a polished single-page landing site:

- `index.html` - page structure and content
- `styles.css` - full responsive design system
- `script.js` - mobile nav and scroll reveal interactions
- `content-research.md` - recovered archived-site content notes and launch caveats

## Design goal

Create a website that feels stronger than a standard contractor-template agency site: modern, premium, mobile-first, fast, and grounded in CAC's actual legacy as a Cincinnati commercial HVAC and design-build mechanical contractor established in 1938.
